# guidance-R-pref
Section preferences on R packages and code, organized by topic below.

# Data frames
Use either data.tables ([data.table package](https://cran.r-project.org/package=data.table)) or tibbles ([tibble package](https://cran.r-project.org/package=tibble) or [readr package](https://cran.r-project.org/package=readr) or the [tidyverse set of packages](https://cran.r-project.org/package=tidyverse)).
These packages follow two quite different approaches.
